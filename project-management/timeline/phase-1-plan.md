# Phase 1 Project Plan: Nurse-Worn Medical Assistant MVP

## Goal:
Develop a functional nurse-worn medical assistant (MVP) with LLM integration for policy Q&A and language translation.

## Duration:
Approximately 4 weeks

## Key Deliverables:
- Wearable client application (buildable image + configuration)
- Edge gateway service with endpoints for ASR/LLM/TTS, session logs, and metering
- Data connectors for Mount Sinai policy retrieval and translation models
- Security & compliance checklist (HIPAA/PHI scope, logging, retention)
- Pilot runbook: deployment, monitoring, incident handling, and rollback

## High-Level Milestones:

### Week 1: Setup & Initial Design
- Project setup and environment configuration.
- Detailed design for wearable client and gateway service.
- Finalize security and compliance requirements.

### Week 2-3: Development & Integration
- Implement wearable client application features.
- Develop edge gateway service components (ASR/LLM/TTS pipeline).
- Integrate data connectors for policy retrieval.

### Week 4: Testing & Documentation
- Comprehensive testing of all MVP components.
- Finalize security & compliance checklist.
- Prepare pilot runbook.
- Prepare for Phase 1 review.

## Team & Effort:
- **Team:** 2 dedicated contributors (engineering + strategic design)
- **Effort:** 80 hours (design & engineering) + 16 hours (strategic consulting)
