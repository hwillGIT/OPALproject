v1 iPhone surfaces · core flow

The most demo-worthy moments — voice surface states, response cards, and the AI-awareness moment that anchors the H6 storyline.

S1.A · HOME
Home — Sarah, 4-South Night Shift
Identity, mode pill, recent queries, quick actions, large TALK button. Entry point for every journey.
 S1.B / C / D / E
Voice command states
Listening (waveform), Processing (spinner + latency badge), Speaking (sync'd transcript), Error (offline). All four states side-by-side.
 S2.A · MODES
Mode switcher modal
Bottom sheet for picking NORMAL / BROADCAST / INTERPRETER with group and language selectors.
 S3.A · RESPONSE
Standard Response Card — Sarah §4.1

 S3.B · AI ROUTING
Response Card with hospital AI pill — Robert §4.6

 S3.C · FIRST-TIME
First-time AI tool exposure

Manager dashboard — all 10 views

Web app for unit managers, CNOs, clinical informatics, and the CCIO. Anchored to H6 (AI awareness gap), H10 (CCIO needs ROI), H13 (silent abandonment), H17/H18 (vendor renewal economics).

D1 · ADOPTION
Adoption overview
"Are nurses using LYNA?" KPI strip + 90d trend + hour-of-day heatmap + per-nurse status pills.
 D2 · AI ROUTING
AI tool routing
CCIO renewal view. Tool cards in 4 health states (healthy / issues / no traffic / coverage low).
 D3 · QUERIES
Top queries (answered + unanswered)
NI team view. "Add to KB" affordance on every unanswered query. Urgent-flag option.
 D4 · LATENCY
Response times
Histogram with 2s threshold line. Per-query-type + per-unit breakdowns. Slow-query log.
 D5 · H13
Silent abandonment alerts
At-risk list with severity pills + reach-out template (Teams / Mattermost / email).
 D6 · BROADCAST
Broadcast / help coverage
Ack-within-30s/60s KPIs. Per-shift comparison. Escalation log.
 D7 · AUDIT
Critical alerts audit
Read-only audit log. Per-event PDF export for the post-event review packet.
 D8 · INTERPRETER
Interpreter sessions
Sessions-by-language with coverage-line divider. Title VI compliance audit PDF.
 D9 · TOOL DETAIL
Per-AI-tool drill-down
Drill from D2. Routing trend, top intents, first-time exposures, issues, ROI vs contract.
 D10 · BUYER ROI
Buyer-facing ROI
CCIO budget packet view. Headline card + sensitivity bounds + I-Corps-anchored calc.
