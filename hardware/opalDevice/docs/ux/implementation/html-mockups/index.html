v1 iPhone surfaces

Six representative surfaces covering the highest-impact moments of the ten LYNA clinical-staff journeys. Each renders inside an iPhone 14 frame (390 × 844 pt).

S1.A · NORMAL
Home — Sarah, 4-South Night Shift
Foundation surface. Identity, mode pill, recent queries, quick actions, large TALK button.
 S3.B · AI ROUTING
Response Card — Robert §4.6
Hospital AI-source pill dramatizes the H6 awareness gap. Names the integrated reference AI; cites multi-source provenance.
 S4.A · BRAIN SHEET
Unit Brain Sheet — Marcus §4.3
28-patient handoff in one scroll. Color-coded sections (issues / NPO / discharges / admits). Replaces 20 minutes of paper-and-chart-scanning.
 S5.A · CRITICAL ALERT
Rapid Response — Sofia §4.7
Full-screen takeover, pulsing red border, large ACK button. Cardiac step-down (4-East), not ICU — per BMC V2 beachhead.
 S6a · BROADCAST
Isolation Broadcast — Priya §4.8
Sender's ack tracker after broadcasting from an isolation room. First-acknowledge-wins; logged for unit responsiveness metrics.
 S7.B · INTERPRETER
Bidirectional Session — Maya §4.10
Active-speaker indicator, bilingual transcript, "queue certified interpreter" affordance. Title VI compliant scope.

Manager dashboard views

Three representative views from the 10-view dashboard spec. Built for desktop / large-tablet at 1440 × 900 minimum.

D1 · ADOPTION
Adoption Overview — Marcus, 4-South Mgr
"Are nurses actually using LYNA?" KPI strip, 90-day trend, per-unit hour-of-day heatmap, per-nurse table with abandonment-watch chips.
 D2 · AI TOOL ROUTING
AI Tool Routing — Dr. Ramirez, CCIO
Most strategically important view for vendor renewals. Per-tool card, coverage, first-time exposures, sparkline trends, "no traffic" + "coverage low" warnings.
 D10 · BUYER ROI
Buyer-facing ROI — CCIO budget packet
Headline card with time-saved estimate; sensitivity bounds; calculation breakdown anchored to I-Corps benchmarks; PDF export footer.
