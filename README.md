# OPAL Project: Nurse-Worn Medical Assistant

This project aims to develop a nurse-worn medical assistant and integrate it with the Edge Micropayment Platform for Integrated Commerce (EMPIC).

## Phases:
- **Phase 1 (MVP):** Wearable assistant + LLM integration (policy Q&A + translation).
- **Phase 2:** EMPIC Payment Integration (Escrow, Metering & Split-Payouts).
